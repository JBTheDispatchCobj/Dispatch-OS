# [Institution] Terminal PRD

## Promise
A living operating environment that curates the market, the institution, the person, and the network into the next best actions.

## Core modules
- Institution profile
- Personal and role profile
- Personalized Auric feed
- Market search
- Network activity
- Products, vendors, startups, and partners
- Cartridge catalog and installed capabilities
- Private documents and memory
- Collaboration
- Relationship workspaces
- Usage and execution

## Paid entry
The first paid tier must be inexpensive enough for an individual executive or employee to purchase without committee approval. Initial hypothesis: approximately $20/month per user, including bounded private memory and usage.

## Expansion
Usage-priced cartridge work, premium deliberation, expert review, integrations, pilots, diligence, administration, and managed services.
