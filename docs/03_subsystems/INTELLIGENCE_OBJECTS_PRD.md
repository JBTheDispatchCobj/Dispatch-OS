# Intelligence Objects PRD

An Intelligence Object is the atomic sourced unit for publication and decision support. It contains facts, claims, affected entities, relevance, confidence, shelf life, recommended action, and renderable variants.

## Required sections for implementation
- Goals and non-goals
- Object contracts
- Inputs and outputs
- Rules and workflows
- Permissions and visibility
- Model usage and escalation
- Metrics and tests
- Failure modes
