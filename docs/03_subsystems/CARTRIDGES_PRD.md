# Cartridges PRD

Cartridges package capability using data contracts, rules, prompts, workflows, metrics, permissions, pricing, evaluation fixtures, and render contracts. Types include domain, role, workflow, relationship, and service cartridges.

## Required sections for implementation
- Goals and non-goals
- Object contracts
- Inputs and outputs
- Rules and workflows
- Permissions and visibility
- Model usage and escalation
- Metrics and tests
- Failure modes
