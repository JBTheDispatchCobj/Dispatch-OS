# Quarterly Business Review and Value Report

## Executive Summary
Objectives, current health, material outcomes, risks, decisions required.

## Adoption
Active users, workflows, completion, approvals, evidence packages, dashboards, and training.

## Value
Baseline, current result, formula, source evidence, confidence, financial and nonfinancial impact.

## Quality and Risk
Data quality, correction rate, incidents, exceptions, stale evidence, security and compliance status.

## Service
Support volume, SLA performance, root causes, roadmap items, and partner performance.

## Expansion
Unmet workflows, readiness, expected value, effort, controls, capacity, and recommended next scope.

## Decisions
Renew, expand, remediate, pause, or exit; named owner and deadline.
