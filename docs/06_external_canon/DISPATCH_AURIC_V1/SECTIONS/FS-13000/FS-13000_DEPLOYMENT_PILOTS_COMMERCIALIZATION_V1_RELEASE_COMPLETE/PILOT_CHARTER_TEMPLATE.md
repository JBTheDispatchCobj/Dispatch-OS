# Pilot Charter Template

## Identity
Pilot ID, institution, executive sponsor, workflow owner, Auric lead, start/end dates.

## Problem and Baseline
Describe the current process, pain, measurable baseline, affected users, and consequences.

## Scope
List included users, objects, workflows, data sources, connectors, agents, outputs, environments, and explicit exclusions.

## Governance
Document classification, security owner, data owner, approval authorities, evidence requirements, model providers, retention, and incident path.

## Delivery
Milestones, RACI, dependencies, training, support model, change process, and budget.

## Success
Define adoption, quality, time, cost, risk, value, and user outcomes with formulas and source evidence.

## Exit and Scale
Define termination, rollback, data disposition, scale criteria, renewal path, and final decision authority.

## Signatures
Executive sponsor; institution security/data authority; workflow owner; Auric implementation lead; commercial authority.
