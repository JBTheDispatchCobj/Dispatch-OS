# FlowPilot AI — Controlled Pilot Board Brief

## Purpose

Evaluate a limited member-service workflow automation pilot using synthetic data and no autonomous member-facing decisions.

## Proposed Scope

- 25 internal users
- one member-service workflow
- sandbox integration
- no final credit, compliance, or legal decisions
- ninety-day duration

## Readiness

Overall readiness: conditionally viable.

Primary blockers:

- production-like test environment not yet available
- model-risk documentation incomplete
- final data-flow review pending

## Required Approvals

- business owner
- technology
- security
- vendor risk
- compliance

## Success Measures

- workflow cycle-time reduction
- user adoption
- error rate
- escalation rate
- compliance exceptions
- support tickets

## Recommendation

Proceed only after the security and data-flow conditions are satisfied.
