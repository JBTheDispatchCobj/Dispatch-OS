# Production Readiness Checklist

- [ ] Release manifest and checksums validated
- [ ] Critical services have accountable owners and SLOs
- [ ] Dashboards, alerts, and escalation paths are active
- [ ] Connector and agent suspension procedures tested
- [ ] Backup and restore tested
- [ ] Rollback tested
- [ ] Security/privacy gate approved
- [ ] Migration rehearsal passed
- [ ] Support and incident teams trained
- [ ] Pilot users and permissions configured
- [ ] Known issues and residual risks accepted
- [ ] Customer and internal communications approved
