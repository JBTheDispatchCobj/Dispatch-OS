# Institutional Agent Library Specification

The CSV catalog defines the V1 workforce. Every production agent must add: Agent ID, owner, domain, authority level, allowed capabilities, prohibited actions, Context Pack schema, required evidence, tools/connectors, model policy, output schema, evaluation rubric, escalation rules, cost ceiling, latency target, privacy class, events, and tests.

Agents are stateless workers. They do not own truth, memory, policy, or business authority.
