# 02 Private Terminal Onboarding

Select institution → confirm authority → configure tenant and roles → import public profile → capture objectives and priorities → approve data sources/connectors → set model and privacy policies → activate cartridges → assemble initial workspaces and reports → acceptance review.

## Required implementation fields

Owner, trigger, inputs, objects, relationships, policies, agents, human approvals, outputs, metrics, failure modes, recovery, events, and acceptance tests.
