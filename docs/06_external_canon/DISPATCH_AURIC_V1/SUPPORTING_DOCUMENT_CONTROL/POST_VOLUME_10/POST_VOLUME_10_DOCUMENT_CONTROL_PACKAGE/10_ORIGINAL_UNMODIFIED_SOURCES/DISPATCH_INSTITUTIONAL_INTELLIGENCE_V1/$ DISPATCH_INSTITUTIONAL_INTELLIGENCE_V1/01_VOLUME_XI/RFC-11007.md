# RFC-11007 — Institutional Profile Assembly

Status: Draft 1.0

Owner: Auric Works

Authority: Dispatch Constitution and Volumes I–X

## Profile types

Institution, executive, fintech/startup, vendor, product, capability, regulator, association, fund/investor, and market.

## Assembly rules

Profiles are dynamic graph projections, never hand-maintained pages. Every profile includes identity, evidence, confidence, freshness, relationships, activity, systems, products, readiness, risks, opportunities, and next actions.

## Public/private split

Public profiles use public and licensed sources. Private profiles add institution-provided systems, documents, preferences, strategy, appetite, and operating memory.
