# RFC-11014 — Institutional Intelligence APIs

Status: Draft 1.0

Owner: Auric Works

Authority: Dispatch Constitution and Volumes I–X

## API families

Profiles, institutions, fintechs, products, capabilities, relationships, readiness, opportunities, pilots, partnerships, investments, regulation, evidence, publications, and configurations.

## Required response fields

Canonical objects, facts, evidence, confidence, freshness, visibility, score explanation, recommended next actions, audit reference, and graph version.

## Rule

Clients request institutional capabilities and outcomes, never raw storage operations.
