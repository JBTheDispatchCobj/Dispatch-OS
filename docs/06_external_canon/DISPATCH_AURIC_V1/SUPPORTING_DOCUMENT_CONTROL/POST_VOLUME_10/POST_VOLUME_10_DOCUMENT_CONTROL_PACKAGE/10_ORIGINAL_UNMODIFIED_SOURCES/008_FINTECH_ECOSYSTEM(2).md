## FinTech Ecosystem

Version: 1.0

## Purpose

This document defines the canonical FinTech ecosystem for Dispatch. It
provides the taxonomy, capability model, operational lifecycle,
institutional buying process, connector priorities, object mappings, and
relationship model for financial technology companies.

# 9. Dispatch Truth Model

Every FinTech is represented as:

Organization + Capability Set + Products + Integrations + Institution
Relationships + Compliance Profile + Commercial Profile + Operational
Metrics + Connector Map + Knowledge Graph

No fintech is modeled solely as a company. Dispatch models what it does,
how it integrates, who buys it, and how it creates value.
