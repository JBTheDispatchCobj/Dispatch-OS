# MVP Acceptance Tests

1. All active U.S. credit unions can be imported and receive a public profile.
2. Every displayed conclusion distinguishes source fact, calculation, inference, and verification.
3. A CU profile updates when a new Call Report is loaded.
4. A startup website/deck produces a sourced company profile and gap list.
5. The system explains two-way CU/startup fit and recommended relationship roles.
6. One Intelligence Object renders differently for CEO, CFO, CIO, risk, and founder cartridges.
7. A Brief can be generated with three to five relevant items.
8. A relationship can move through interest, discussion, pilot, integration, partnership, investment, and monitoring states.
9. Model routing respects task risk, cost ceiling, and confidence.
10. Private tenant facts never appear in another tenant's output without permission.
